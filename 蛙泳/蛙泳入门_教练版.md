## 🎯 教学核心理念

| 原则 | 说明 |
|------|------|
| **水性第一** | 不着急教动作，先让学员"不怕水、享受水"，这是所有后续学习的基础 |
| **感觉驱动** | 强调"找感觉"，而不是机械模仿动作；动作要对，但感觉更重要 |
| **分解细化** | 每个动作拆到不能再拆，循序渐进；慢即是快，基础决定高度 |
| **纠错前置** | 提前告诉学员"会犯什么错"，预判问题比纠正错误更重要 |

> 🧠 **教练心法**：教学员，而不是教学动作。先让学员热爱水，剩下的交给时间。

## 第1课：关于蛙泳对游泳初学者的作用

> 视频：P1

**[教学目标]**
- 理解蛙泳的特点和适用场景
- 建立正确的学习预期和心理准备
- 了解蛙泳"节奏可控、抬头呼吸、不容易呛水"的优势

**[教学内容]**

### 1.1 蛙泳的特点（10分钟）
- 蛙泳是**唯一以腿部动作为主要推进**的泳姿（其他三种都以手臂为主）
- 蛙泳的节奏：**划手换气 → 收腿 → 蹬腿 → 漂一会儿**
- 每个周期有明确的"停歇点"——漂的时候可以休息

### 1.2 为什么初学者从蛙泳开始？（15分钟）
- **抬头呼吸**：永远朝前看，不容易呛水
- **节奏可控**：想停就停，想走就走
- **节省体力**：不需要一直踢腿打水
- **用途广泛**：实用性强——会蛙泳可以救人、自救、游长距离

### 1.3 学习预期建立（10分钟）
- 告诉学员：蛙泳入门预计需要**8-12课时**（因人而异）
- 强调：每个人的进度不一样，不要和别人比
- 心理建设：**学会游泳需要时间，但每节课都会有进步**

**[时间分配]**
- 蛙泳特点讲解：10分钟
- 为什么从蛙泳开始：15分钟
- 学习预期建立：10分钟
- 问答互动：10分钟

**[课后作业]**
- 回家练习：在洗脸盆里憋气，鼻子慢慢吐泡泡
- 心理建设：想象自己在水中漂浮的美好感觉

**[教学提示]**
- 这一课看似"没教东西"，但心理建设非常重要
- 让学员知道游泳不难，但需要时间；让他们期待而不是畏惧
- 如果是成人学员，可以讲讲蛙泳的历史和有趣知识

## 第2课：超级怕水怎么办？

> 视频：P2

**[教学目标]**
- 消除或缓解对水的恐惧
- 建立在水中的安全感
- 找到"不怕水"的开关

**[教学内容]**

### 2.1 水中行走（15分钟）
- 水深到胸口，慢慢走
- 向前走→向后走→侧向走→转圈圈
- 💡 **感觉**：感受水的阻力，脚步要踏实
- ⚠️ **重点**：不要急，一步一步来，水会托着你

### 2.2 憋气练习（20分钟）
- **第一步**：盆水憋气——用冷水，效果更好
- **第二步**：水龙头下憋气——感受水流刺激
- **第三步**：水中憋气——捏住鼻子，嘴巴张开发"啊"声
- 💡 **感觉**：水不会伤害你，放松，放松
- ⚠️ **绝对不能用鼻子吸气**，只能用嘴巴吸气

### 2.3 心理建设（10分钟）
- "水不会伤害你"——这是核心理念
- "放松身体，水会托着你"——信任水的浮力
- 教练或同伴在旁边：**"有人会保护你"**——安全感建立
- 告诉学员：**如果不舒服，随时可以站起来**

### 2.4 小朋友特别建议（5分钟）
- 用游戏的方式：吹泡泡、吐口水、水枪战
- 不要强迫，尊重节奏
- 每次进步一点点，不要急于求成

**[时间分配]**
- 水中行走：15分钟
- 憋气练习：20分钟
- 心理建设：10分钟
- 小朋友特别建议：5分钟（如果有小朋友）

**[纠错提示]**
| 学员表现 | 纠正方法 |
|------|------|
| 学员不敢下蹲 | 先从浅水区开始，绝对不要强迫 |
| 憋气就慌 | 先练呼气：嘴巴吸气、鼻子呼气，慢慢来 |
| 紧紧抓住池边不放 | 先让他站在池边玩水，不急着松手 |
| 一进水就紧张 | 先在池边洗脸、撩水，适应湿润感 |

**[课后作业]**
- 在家：洗脸盆憋气练习，每天3次，每次5组（憋气10秒+休息5秒）
- 下次课目标：能在水中憋气15秒以上

**[教学提示]**
- 这一课可能需要1-2次课才能完成，不要赶进度
- 学员怕的不是水，是"不知道会发生什么"——所以要提前告知每个步骤
- 多用"我们试试"，少用"你必须"

## 第3课：换气——基本练习

> 视频：P3

**[教学目标]**
- 建立正确的呼吸节奏：吸→憋→呼
- 掌握"嘴巴吸气、鼻子呼气"的基本模式
- 为后续水中换气打下基础

**[教学内容]**

### 3.1 呼吸节奏建立（陆上练习，15分钟）
- **节奏练习**：吸3秒→憋5秒→呼5秒
- 💡 **感觉**：像闻花香一样吸气，像吹蜡烛一样呼气
- 教练喊口令："吸——憋——呼——"
- 先在岸上练习10组，形成肌肉记忆

### 3.2 嘴巴吸气、鼻子呼气（10分钟）
- ⚠️ **绝对不能用鼻子吸气**，否则必呛水
- 练习：嘴巴张开吸气 → 捏住鼻子 → 鼻子慢慢吐气
- 💡 **感觉**：吐气的时候像"喝汤"，气体从鼻子后面出来

### 3.3 水中憋气进阶（20分钟）
- **第一步**：捏住鼻子，嘴巴张开发"啊"——感受气体从嘴里冒出来
- **第二步**：不捏鼻子，嘴巴张开，头浸入水中，鼻子慢慢吐气
- **第三步**：逐渐增加浸水深度，从下巴到嘴巴到鼻子
- 💡 **感觉**：气泡从鼻子出来，像小鱼在吐泡泡
- ⚠️ **如果不舒服，立刻抬头**，不要勉强

### 3.4 "水不会伤害你"再次强化（5分钟）
- 告诉学员：水中的感觉是正常的，不要怕
- 鼓励：做得很好，继续

**[时间分配]**
- 呼吸节奏建立：15分钟
- 嘴巴吸气鼻子呼气：10分钟
- 水中憋气进阶：20分钟
- 心理建设：5分钟

**[纠错提示]**
| 学员表现 | 纠正方法 |
|------|------|
| 用鼻子吸气 | 立刻让他停，强迫：只能嘴巴吸气 |
| 吸气太急 | 让他"闻花香"——慢慢吸气，不要急促 |
| 憋气时身体紧绷 | 让他肩膀放松，提醒"放松，水会托着你" |
| 不敢把脸浸水里 | 先从撩水开始，再洗脸，再浸嘴，再浸鼻子 |

**[课后作业]**
- 在家练习：对着镜子练"嘴巴吸气、鼻子呼气"
- 目标：能自然做到"口吸鼻呼"，不混淆

**[教学提示]**
- 换气是一切的基础，这一课再怎么强调都不为过
- 很多学员学游泳多年还在呛水，根源就是这一课没学好
- 要让学员建立"呼吸通道"的正确概念：上面是嘴，下面是鼻子，不能串

## 第4课：换气——浅水区扶池边练习

> 视频：P4

**[教学目标]**
- 在浅水区实际体验水中换气
- 掌握"抬头张嘴吸气、低头鼻子呼气"的节奏
- 建立水中换气的信心

**[教学内容]**

### 4.1 浅水区行走 + 适应（10分钟）
- 继续第一课的水中行走练习
- 双手扶池边，水深到胸口，慢慢走
- 感受浮力：身体变轻了

### 4.2 扶池边换气练习（20分钟）
- **预备姿势**：双手扶池边，身体浸入水中，下巴在水面下
- **练习一**：抬头张嘴吸气 → 低头鼻子呼气
- **练习二**：连续练习：吸一口气→低头吐泡泡→抬头吸气→低头吐泡泡
- **练习三**：每组10次，休息10秒，重复3组
- 💡 **感觉**：抬头的时候嘴巴像"张开的鱼"，低头的时候鼻子像"洒水壶"
- ⚠️ **重点**：吸气必须抬头，不能在水下用嘴吸气

### 4.3 计时憋气换气（15分钟）
- 吸一口气，憋住，低头入水，鼻子慢慢吐气
- 计时：15秒→20秒→25秒
- 💡 **感觉**：时间够了再抬头，不要着急

### 4.4 安全确认（5分钟）
- 再次强调：不舒服就立刻抬头
- 告诉学员：**池边永远在你手边**，随时可以扶

**[时间分配]**
- 浅水区适应：10分钟
- 扶池边换气：20分钟
- 计时憋气：15分钟
- 安全确认：5分钟

**[纠错提示]**
| 学员表现 | 纠正方法 |
|------|------|
| 低头时用嘴呼气 | 让他捏住鼻子，强迫只用鼻子呼气 |
| 抬头时还在吐气 | 告诉他"抬头前把气吐完，抬头只吸气" |
| 吸气吸到水 | 强调"嘴张开就好，不要吸太深" |
| 身体往下沉 | 手适当撑池边，身体放松，嘴巴保持在水面上 |

**[课后作业]**
- 在家练习：对着水盆练"抬头吸气、低头呼气"30次
- 下次课目标：能连续完成10次完整的换气循环

**[教学提示]**
- 这是从"不怕水"到"会呼吸"的关键一步
- 教练要一直在旁边，让学员知道"有人看着我，很安全"
- 可以让学员互相看着练习，形成轻松的氛围

## 第5课：连续换气练习

> 视频：P5

**[教学目标]**
- 掌握连续换气的节奏
- 能连续做"抬头吸气、低头呼气"至少10次以上
- 为后续"踩水"和"漂浮"打下呼吸基础

**[教学内容]**

### 5.1 连续换气节奏练习（浅水区，20分钟）
- **练习一**：吸一口气→低头吐泡泡（数3秒）→抬头吸气（嘴巴张开）→低头吐泡泡
- **练习二**：连续10次，组间休息10秒，做3组
- **练习三**：逐渐加快节奏，但保持质量
- 💡 **感觉**：像"小鸭子在水面"，一上一下，自然流畅
- ⚠️ **关键**：每个动作都要完整——吸气要吸饱，呼气要呼尽

### 5.2 换气与行走结合（15分钟）
- 扶池边练习：行走的同时做换气
- 向前走：低头时走一步，抬头时走一步
- 侧向走：体验不同方向的换气
- 💡 **感觉**：换气时身体不要前倾，保持直立

### 5.3 脱离池边的连续换气（15分钟）
- 手离开池边，在浅水区站立
- 做连续换气，同时轻轻原地踏步
- 💡 **感觉**：脚踩池底，身体直立，嘴巴一直在水面上

### 5.4 建立信心（5分钟）
- 告诉学员："你已经掌握了水中呼吸的基础"
- 鼓励："接下来我们要学漂浮，这是游泳最重要的技能"

**[时间分配]**
- 连续换气节奏：20分钟
- 换气与行走：15分钟
- 脱离池边练习：15分钟
- 信心建立：5分钟

**[纠错提示]**
| 学员表现 | 纠正方法 |
|------|------|
| 换气节奏乱 | 喊口令："吸——呼——吸——呼"，配合节奏 |
| 吸气不充分 | 让他多吸一点，"像闻花香一样吸满" |
| 身体前倾 | 提醒"身体站直，肩膀放松" |
| 紧张导致换气变快 | 让他暂停，深呼吸，重新开始 |

**[课后作业]**
- 在家练习：对着水盆连续做20次换气，体会节奏
- 下次课目标：能连续换气15次以上不停顿

**[教学提示]**
- 这一课要把换气练到"不用想就会"的程度
- 呼吸是游泳的根基，呼吸不顺畅，后续所有动作都会乱
- 达到"连续换气15次"才算合格，才能进入下一课

## 第6课：漂浮和漂浮站立

> 视频：P6

**[教学目标]**
- 掌握抱膝漂浮（团身漂浮）的动作
- 能在水中漂浮10秒以上
- 掌握从漂浮到站立的正确顺序

**[教学内容]**

### 6.1 抱膝漂浮讲解（5分钟）
- 目的：让身体像"气球"一样飘起来
- 口诀：**低头、抱膝、团成一团、像球一样**
- 关键点：头往下扎，屁股往上翘，整个人蜷起来

### 6.2 陆上模仿练习（10分钟）
- 岸上趴着练：像青蛙一样蜷缩
- 练习：低头看膝盖，膝盖往胸口收
- 💡 **感觉**：像虾一样弓着背

### 6.3 水中抱膝漂浮练习（20分钟）
- **练习一**：扶池边，低头入水，感受浮力
- **练习二**：手松开，慢慢蜷起来，感受"漂"的感觉
- **练习三**：自己尝试漂，教练在旁边保护
- 💡 **感觉**：水从身体两侧流过，身体很轻，像气球飘在空中
- ⚠️ **常见错误**：身体不团、头不低、屁股不翘

### 6.4 漂浮站立（15分钟）
- ⚠️ **这是本课最关键的技能**
- 口诀：**先收手——再收腿——脚踩地——站起来**
- **错误示范**：先收腿，容易乱，容易呛水
- 分解动作：
  1. 漂浮状态
  2. 双手从两侧收回来，抱住膝盖（手先动）
  3. 膝盖往胸口收，身体变短（腿后动）
  4. 脚踩池底（或者往下踩）
  5. 站稳，站起来
- 💡 **感觉**：像蜗牛缩进壳里，再伸出来

**[时间分配]**
- 抱膝漂浮讲解：5分钟
- 陆上模仿：10分钟
- 水中漂浮练习：20分钟
- 漂浮站立：15分钟

**[纠错提示]**
| 学员表现 | 纠正方法 |
|------|------|
| 身体平着漂不起来 | 让他"低头、弓背、屁股翘"，必须蜷成一团 |
| 漂浮时身体旋转 | 让他手臂夹紧，头埋进水里 |
| 站立时先收腿 | 强调"手先动，再收腿"，重复喊口令 |
| 站立时呛水 | 让他在浅水区练，水到胸口，站起来方便 |

**[课后作业]**
- 在家练习：对着水盆练习"蜷缩——展开"的循环
- 下次课目标：能独立完成漂浮+站立，不靠池边

**[教学提示]**
- 漂浮是游泳的"灵魂技能"，学会了漂浮就等于学会了50%的游泳
- 这一课要让学员体验"被水托着"的感觉，建立对水的信任
- 漂浮站立比漂浮更难，要多练"手先动"的口令
- 安全第一：教练要站在学员旁边，随时准备扶

## 第7课：抬头换气

> 视频：P7

**[教学目标]**
- 掌握"抬头换气"的方法
- 能做到"低头呼气、抬头吸气"
- 为蛙泳腿+呼吸配合做准备

**[教学内容]**

### 7.1 抬头换气原理（5分钟）
- 目的：在蛙泳中，腿打水时身体在水下，需要抬头吸气
- 关键：**嘴巴在水面上，鼻子在水下**

### 7.2 水中站立抬头换气（15分钟）
- 站在浅水区，水到胸口
- **练习一**：深吸一口气，低头，鼻子慢慢吐泡泡 → 抬头，嘴巴张开吸气
- **练习二**：连续10次，节奏：吸3秒→憋1秒→呼3秒→抬头吸
- **练习三**：每组15次，休息10秒，做3组
- 💡 **感觉**：抬头的时候嘴巴像"张开的碗"，等着接空气

### 7.3 扶池边抬头换气（15分钟）
- 双手扶池边，身体浸入水中
- 低头吐泡泡 → 抬头张嘴吸气 → 低头吐泡泡
- ⚠️ **关键**：抬头的时候不要呼气，要吸气；低头的时候才呼气

### 7.4 蹬边滑行+抬头换气（10分钟）
- 蹬边滑行出去，滑到一半时：
- 抬头张嘴吸气 → 低头继续滑行
- 练习5-8次
- 💡 **感觉**：滑行的时候身体很流线，抬头的时候像"露出水面的小恐龙"

**[时间分配]**
- 抬头换气原理：5分钟
- 水中站立练习：15分钟
- 扶池边练习：15分钟
- 蹬边+抬头：10分钟

**[纠错提示]**
| 学员表现 | 纠正方法 |
|------|------|
| 抬头时还在呼气 | 强调"抬头只吸气，低头才呼气"，喊口令练习 |
| 抬头太高，身体下沉 | 让他"下巴往上抬，嘴巴张开就好" |
| 吸气吸到水 | 让他把嘴张到最大，"像河马一样大嘴" |
| 换气节奏乱 | 用口令："吸——憋——呼——"，慢慢来 |

**[课后作业]**
- 在家练习：对着水盆练抬头换气，每组20次，每天3组
- 下次课目标：蹬边滑行+抬头换气能完成5次以上

**[教学提示]**
- 抬头换气是蛙泳的必备技能，因为蛙泳需要低头打水
- 这一课要为下一课的"漂浮蹬腿+换气"做铺垫
- 告诉学员：抬头换气是"不得已"的技能，能不抬头就不抬头（抬头会增加阻力）

## 第8课：如何在漂浮中控制身体重心

> 视频：P8

**[教学目标]**
- 理解"身体重心"的概念
- 能在漂浮中控制身体保持水平
- 解决"脚往下沉"的问题

**[教学内容]**

### 8.1 重心概念讲解（5分钟）
- 陆上讲解：人的重心在肚脐附近
- 在水中：如果重心在水面以上，脚就会沉下去
- 目标：**让重心保持在水面以下，身体才能水平**

### 8.2 脚往下沉的原理（10分钟）
- 为什么脚会沉？因为肺部有气，重心在上半身
- 💡 **感觉**：像秤一样，重的一头会沉下去
- 解决方案：**吐气——把肺部的气吐掉一部分**

### 8.3 吐气让身体水平（15分钟）
- **练习一**：深吸一口气漂浮 → 慢慢把气吐出来 → 观察身体变化
- **练习二**：漂浮后，鼻子缓缓吐气 → 脚会慢慢浮上来
- **练习三**：每次漂浮前先吐一口气，保持"气往下走"的感觉
- 💡 **感觉**：气从肚子里往下走，像水流到脚那边

### 8.4 重心调整练习（15分钟）
- **练习一**：双手往下压水——借力让脚浮起来
- **练习二**：头往上抬——脚会往下沉（反向验证）
- **练习三**：身体成一字形时，双手轻轻往下压水，控制平衡
- 💡 **感觉**：像天平一样，一边压下去，另一边就浮上来

### 8.5 连续漂浮+重心控制（10分钟）
- 蹬边滑行出去 → 漂浮 → 控制身体水平 → 站立
- 目标：能滑出5米以上，身体保持水平
- ⚠️ **常见错误**：憋太多气、身体蜷缩、头抬太高

**[时间分配]**
- 重心概念：5分钟
- 脚沉原理：10分钟
- 吐气练习：15分钟
- 重心调整：15分钟
- 连续练习：10分钟

**[纠错提示]**
| 学员表现 | 纠正方法 |
|------|------|
| 脚一直沉 | 让他"深吸一口气，然后慢慢吐完" |
| 身体蜷成一团 | 让他"身体伸直，手臂夹耳，头放低" |
| 头抬太高 | 让他"眼睛往下看，鼻子往下看" |
| 漂浮不稳 | 让他"双手往下压水，像压秤一样" |

**[课后作业]**
- 思考：为什么梦觉说要"吐气让身体水平"？
- 下次课目标：蹬边漂浮能滑出5米以上，身体接近水平

**[教学提示]**
- 这是"水性"好坏的重要标志——能控制身体在水中的位置
- 水感好的学员会自己找到感觉，水感差的需要多引导
- 这一课要反复强调"吐气"的重要性，憋气是漂浮最大的敌人

## 第9课：蹬边漂浮

> 视频：P9

**[教学目标]**
- 掌握蹬边滑行的正确姿势
- 能在蹬边后保持流线型身体，滑行5米以上
- 为蛙泳腿练习打下基础（蹬边后打腿）

**[教学内容]**

### 9.1 蹬边姿势讲解（5分钟）
- 面对池壁，站在池边
- 准备姿势：膝盖弯曲，手扶池边
- ⚠️ **常见错误**：站太直、离墙太远、手离开池边太早

### 9.2 蹬边滑行动作分解（15分钟）
- **第一步**：深吸一口气，低头
- **第二步**：双手抓住池边，膝盖弯曲，像青蛙一样蹲着
- **第三步**：脚后跟往池壁靠，膝盖在肩膀前方
- **第四步**：手往前伸，身体趴平，手臂夹耳
- **第五步**：脚用力蹬出去，身体像"箭"一样射出去
- 💡 **感觉**：像弹簧一样弹出去，身体要"直成一条线"

### 9.3 蹬边滑行练习（20分钟）
- **练习一**：正常蹬边滑行，滑到自然停下
- **练习二**：蹬边后身体成一字形，双手往前伸
- **练习三**：蹬边后手指轻轻拨水，感受水流从身体流过
- ⚠️ **重点**：
  - 身体要直，**头不能抬**
  - 手臂要夹紧，**不能张开**
  - 脚蹬出去后要并拢，**不能分叉**

### 9.4 蹬边漂浮站立（10分钟）
- 蹬边滑行出去 → 滑行3-5米 → 漂浮站立
- 口诀：滑到一半 → 双手抱膝 → 脚踩地 → 站起来
- 💡 **感觉**：滑到最远的时候，身体开始往下沉，这时候准备站

**[时间分配]**
- 蹬边姿势讲解：5分钟
- 动作分解：15分钟
- 蹬边滑行练习：20分钟
- 蹬边漂浮站立：10分钟

**[纠错提示]**
| 学员表现 | 纠正方法 |
|------|------|
| 身体塌腰 | 让他"屁股夹紧，肩膀往下压" |
| 头抬太高 | 让他"下巴收起来，眼睛往下看" |
| 脚蹬没力 | 让他"膝盖多弯一点，蹬的时候用力蹬" |
| 滑不远 | 让他"多蹬一下，身体再直一点，手夹紧" |

**[课后作业]**
- 在家：想象蹬边漂浮的动作，练习"身体成一字形"
- 下次课目标：蹬边漂浮能滑出5米以上，身体接近水平

## 第10课：蛙泳腿的基础动作练习

> 视频：P10

**[教学目标]**
- 掌握蛙泳腿"收→翻→蹬→夹→飘"五步口诀
- 能在岸上正确模仿蛙泳腿动作
- 理解翻脚是蛙泳腿最关键的技术

**[教学内容]**

### 10.1 蛙泳腿五步口诀（10分钟）
- 口诀：**收 → 翻 → 蹬 → 夹 → 飘**
- 每一步讲解清楚，让学员建立动作框架
- 💡 **感觉记忆**：
  - 收：像青蛙收腿
  - 翻：像脚踩书页翻起来
  - 蹬：像踹一面看不见的墙
  - 夹：像憋尿一样的力量
  - 飘：像在水里飘一会儿

### 10.2 陆上模仿练习（池边，25分钟）
- **练习一**：趴在池边，手臂伸直，抬头看前方
- **练习二**：腿在水中慢慢做"收"的动作
- **练习三**：做"翻"的动作——脚尖外翻，脚心朝后
- **练习四**：做"蹬"的动作——脚往后蹬
- **练习五**：做"夹"的动作——脚并拢
- **练习六**：做"飘"的动作——腿伸直，漂一会儿
- ⚠️ **重点**：翻脚是技术核心，翻不好，蹬腿就不走水

### 10.3 教练辅助（10分钟）
- 教练用手轻压学员脚踝，感受翻脚的方向
- 学员做动作，教练感受力量是否正确
- 💡 **感觉**：脚踝要放松，像"扇子"一样翻开

**[时间分配]**
- 口诀讲解：10分钟
- 陆上模仿：25分钟
- 教练辅助：10分钟

**[纠错提示]**
| 学员表现 | 纠正方法 |
|------|------|
| 收腿膝盖分太开 | 让他"膝盖并拢，脚后跟往屁股收" |
| 翻脚翻不够 | 让他"脚尖往外拐，脚心对着后方" |
| 蹬腿力量小 | 让他"想象在踹一面墙，用力蹬出去" |
| 夹腿不并拢 | 让他"蹬完立刻夹，像剪刀一样并拢" |

**[课后作业]**
- 在家：趴在床边练习蛙泳腿，体会"翻脚"的感觉
- 下次课目标：能在水中做出完整的"收→翻→蹬→夹"动作

**[教学提示]**
- 蛙泳腿是蛙泳最重要的技术，腿学不好，手和呼吸都白搭
- 这一课至少要花1-2次课来巩固，不要急着往下走
- 翻脚是难点，翻好了就成功了一半

## 第11课：漂浮蹬腿（蛙泳腿）

> 视频：P11

**[教学目标]**
- 能在漂浮状态下做蛙泳腿
- 掌握"蹬一下、漂一会儿"的节奏
- 能前进但不沉

**[教学内容]**

### 11.1 准备姿势（5分钟）
- 扶池边，身体趴下，头浸入水中
- 深吸一口气，身体漂起来
- 腿在身后伸直，蹬边准备

### 11.2 漂浮蹬腿分解练习（20分钟）
- **练习一**：漂浮→收腿→翻脚（不蹬）→伸直（感受翻脚）
- **练习二**：漂浮→收腿→翻脚→蹬→夹→漂
- **练习三**：蹬一次，漂一会儿，感受"走水"
- 💡 **感觉**：蹬完腿之后，身体往前滑，像"船在水面滑行"
- ⚠️ **重点**：蹬腿后要"漂"，不能连续蹬，漂着的时候感受水

### 11.3 连续蹬腿练习（20分钟）
- **练习一**：蹬2次，漂一会儿
- **练习二**：蹬3次，漂一会儿
- **练习三**：连续蹬腿（每组10次，中间休息）
- ⚠️ **常见错误**：一直蹬，不漂——这样会很累，也走不远
- 💡 **感觉**：蹬一次腿，能滑出1-2米，滑到最远的时候再做下一次

### 11.4 浮板辅助蹬腿（10分钟）
- 双手扶浮板，身体趴下
- 做蛙泳腿练习
- 💡 **感觉**：浮板帮助你保持上半身稳定，专注于腿的动作

**[时间分配]**
- 准备姿势：5分钟
- 分解练习：20分钟
- 连续练习：20分钟
- 浮板辅助：10分钟

**[纠错提示]**
| 学员表现 | 纠正方法 |
|------|------|
| 脚往下沉 | 让他"多蹬一下，蹬完漂一会儿，蹬的时候用力" |
| 收腿时身体乱 | 让他"手臂夹紧，头埋低，身体成一字形" |
| 蹬腿不走水 | 感受翻脚是否到位，蹬夹是否连贯 |
| 一直蹬不停 | 喊口令："蹬——漂——蹬——漂" |

**[课后作业]**
- 在家：想象漂浮蹬腿的动作，练习"蹬一下漂一会儿"的节奏
- 下次课目标：扶浮板蹬腿能连续游15米以上

**[教学提示]**
- "蹬一下漂一会儿"是蛙泳腿的核心理念
- 不要让学员一直蹬，要让他们感受"滑行"的价值
- 蹬腿不走水的原因90%是翻脚不到位，要重点强调

## 第12课：蛙泳腿和换气的配合

> 视频：P12

**[教学目标]**

- 口诀：**低头打水，抬头吸气**

### 12.2 分解配合练习（20分钟）
- **练习一**：漂浮，头浸入水中，做一次蛙泳腿，漂一会儿

- ⚠️ **重点**：抬头吸气只吸一次，然后继续打腿

### 12.4 扶板换气练习（10分钟）
- 双手扶浮板，打蛙泳腿
- 每打2-3次腿，抬头换气一次
- 💡 **感觉**：换气的时候身体像"跷跷板"，一头上来一头下去

**[时间分配]**
- 配合原理：5分钟
- 分解练习：20分钟
- 连续配合：20分钟
- 扶板换气：10分钟

**[纠错提示]**
| 学员表现 | 纠正方法 |
|------|------|
| 抬头时腿乱蹬 | 让他"抬头的时候腿伸直，不要动" |
| 换气时身体下沉 | 让他"下巴往上抬，嘴巴张开吸" |
| 吸气吸到水 | 让他"嘴巴张到最大，像河马一样" |
| 低头时用嘴吸气 | 强调"只能用鼻子呼气，低头用鼻子" |

## 第13课：如何控制前进方向？

> 视频：P13

**[教学目标]**
- 理解"不走水"的原因
- 掌握让身体往前走的技术要点
- 能做到"游直线"而不是"画圈圈"

**[教学内容]**

### 13.1 "不走水"的原因分析（10分钟）
- **原因一**：翻脚不到位——脚没有铲到水
- **原因二**：蹬夹不连贯——蹬和夹之间有停顿
- **原因三**：夹水没夹紧——蹬完没有并拢
- **原因四**：身体不流线——手臂或头位置不对

### 13.2 翻脚强化练习（15分钟）
- 站在浅水区，双手扶池壁
- 单腿做翻脚练习：收腿→翻脚→蹬→夹
- 教练检查脚踝是否放松、角度是否正确
- 💡 **感觉**：脚踝像"门轴"一样灵活，翻的时候脚心像"勺子"铲水

### 13.3 蹬夹连贯练习（15分钟）
- 陆上练习：趴在池边，感受"蹬——夹"是连续动作
- 水中练习：慢动作做蛙泳腿，重点是"夹"的力量
- 💡 **感觉**：蹬完立刻夹，中间不能有停顿，像"剪刀合上"一样

### 13.4 身体流线型练习（10分钟）
- 蹬边漂浮，检查身体是否"直成一条线"
- 手臂夹耳，头放低，肩膀放松
- 身体不应该有"塌腰"或者"翘屁股"的现象

### 13.5 方向感知练习（5分钟）
- 在泳池里游10米，用泳池线作为参照
- 观察自己是游直的还是歪的
- 歪的话，检查哪边肩膀更高

**[时间分配]**
- 原因分析：10分钟
- 翻脚强化：15分钟

## 第14课：如何预防蛙泳腿的伤痛？

> 视频：P14

**[教学目标]**
- 了解蛙泳腿可能造成的伤痛
- 掌握正确的热身和放松方法
- 学会科学的练习节奏

**[教学内容]**

### 14.1 蛙泳腿可能造成的伤痛（5分钟）
- **膝盖伤痛**：最常见，膝盖内侧压力过大
- **脚踝伤痛**：翻脚角度太大或者脚踝僵硬
- **腰部伤痛**：塌腰导致腰部压力过大

### 14.2 伤痛原因分析（10分钟）
- **膝盖痛的原因**：
  - 膝盖外翻太大
  - 蹬腿的时候膝盖先动
  - 练习强度太大
- **脚踝痛的原因**：
  - 翻脚角度过大（超过45度）
  - 脚踝没有充分活动
- **腰痛的原因**：
  - 塌腰，骨盆翻转
  - 头部抬太高

### 14.3 热身动作（10分钟）
- 陆上热身：踝关节环绕、膝关节环绕、髋关节旋转
- 水中热身：慢慢走，踢踢腿，泡泡水
- ⚠️ **重点**：下水前一定要热身，不要直接开始打腿

### 14.4 正确练习节奏（10分钟）
- 练习强度：每组不超过25米，中间休息30秒
- 不要"一口气打到底"，要"打几下，飘一会儿"
- 💡 **感觉**：累了就休息，不要硬撑

### 14.5 放松拉伸（5分钟）
- 上岸后：拉伸小腿、拉伸大腿后侧、拉伸髋关节
- 水中放松：仰漂1-2分钟，让身体完全放松

**[时间分配]**
- 伤痛类型：5分钟
- 原因分析：10分钟
- 热身：10分钟
- 练习节奏：10分钟
- 放松拉伸：5分钟

**[教学提示]**
- 这一课很重要，好的教练要保护学员的身体
- 游泳本应该是健康的运动，不要因为错误练习导致伤痛
- 发现学员有不适，立刻停止练习，让他休息

## 第15课：仰头换气

> 视频：P15

**[教学目标]**
- 掌握"仰头换气"的方法
- 理解仰漂的感觉
- 为"抬头蛙"打下基础

**[教学内容]**

### 15.1 仰漂练习（15分钟）
- **练习一**：慢慢往后躺，双手往两侧打开，掌心朝上
- **练习二**：仰漂的时候耳朵在水面下，嘴巴在水面上
- **练习三**：深吸一口气，保持身体放松，体会"漂"的感觉
- 💡 **感觉**：像躺在水床上，身体很放松，水托着后背
- ⚠️ **重点**：不要憋气，要呼气（从鼻子或者嘴）

### 15.2 仰头换气练习（15分钟）
- 仰漂状态下：
- 嘴巴张开吸气 → 鼻子吐气
- 或者：鼻子吐气 → 嘴巴吸气
- 💡 **感觉**：嘴巴像"碗"一样张开，等着接空气

### 15.3 仰漂站立（10分钟）
- **练习一**：仰漂 → 屈膝 → 收腿 → 脚踩地 → 站起来
- **练习二**：仰漂 → 双手往下压水 → 收腿 → 站起来
- ⚠️ **关键**：先搞清楚站立的方向（脚往哪边放）

### 15.4 仰头换气与蛙泳（5分钟）
- 讲解：在蛙泳中，如果想游得更轻松，可以偶尔仰头换气
- 用途：休息、调整节奏、观察前方

**[时间分配]**
- 仰漂练习：15分钟
- 仰头换气：15分钟
- 仰漂站立：10分钟
- 用途讲解：5分钟

**[纠错提示]**
| 学员表现 | 纠正方法 |
|------|------|
| 仰漂时沉下去 | 让他"嘴巴张开多吸点气，肺部充满空气" |
| 耳朵进水 | 让他"头往一边歪，耳朵在水里没关系" |
| 不敢往后躺 | 让他"先坐在池边，往后靠，再慢慢躺下去" |
| 站立时呛水 | 让他"先抬起下巴，再收腿，脚往下踩" |

## 第16课：仰漂站立和转圈圈

> 视频：P16

**[教学目标]**
- 熟练掌握仰漂站立

- 增强水中身体控制能力

**[教学内容]**

- 仰漂 → 双手往头顶方向压水 → 屈膝收腿 → 脚踩地 → 站起来

- **练习三**：仰漂转半圈，然后站起来

- 💡 **感觉**：像躺在床上，看天空，很放松

### 16.4 综合练习（10分钟）

- 连续练习5次

**[时间分配]**

- 仰漂转圈：15分钟
- 放松练习：10分钟
- 综合练习：10分钟

**[教学提示]**
- 仰漂站立是"保命技能"，一定要让每个学员都会
- 仰漂是最舒服的水中姿势，可以帮助学员建立对水的信任
- 这一课可以作为"趣味课"，让学员在玩水中学技能

## 第17课：初级踩水

> 视频：P17

**[教学目标]**
- 理解踩水的概念和用途
- 掌握"简单的踩水"动作
- 为后续学习"抬头蛙"打下基础

**[教学内容]**

### 17.1 踩水概念（5分钟）
- 什么是踩水：身体直立，头在水面上，脚在水下踩踏
- 用途：休息、观察、调整呼吸、在水中保持位置

### 17.2 简单踩水动作（20分钟）
- **姿势**：身体直立，肩膀在水面下
- **手**：双手在胸前往下压水
- **脚**：像走路一样，脚往下踩，膝盖稍微弯曲
- 💡 **感觉**：脚像"踩自行车"，但踩的是水，不是踏板
- 口诀：**手下压，脚下踩，身体直立头在上**

### 17.3 踩水练习（20分钟）
- **练习一**：双手扶池壁，脚踩水
- **练习二**：离开池壁，手扶浮板，脚踩水
- **练习三**：手离开浮板，独立踩水30秒
- **练习四**：踩水10秒，休息5秒，重复5组
- ⚠️ **重点**：头要一直在水面上，不能沉下去

### 17.4 踩水与呼吸（5分钟）
- 踩水的时候：用嘴吸气，用鼻子或者嘴吐气
- 保持呼吸节奏，不要憋气

**[时间分配]**
- 踩水概念：5分钟
- 动作讲解：20分钟
- 踩水练习：20分钟
- 呼吸节奏：5分钟

**[纠错提示]**
| 学员表现 | 纠正方法 |
|------|------|
| 头沉下去 | 让他"脚踩快点，手往下压重点" |
| 身体往前倾 | 让他"屁股收紧，身体直立" |
| 手乱动 | 让他"手在胸前画圈，往下压水" |
| 脚乱踢 | 让他"脚像走路一样，一上一下，不要太快" |

### 18.1 前滚翻概念（5分钟）

## 第19课：蛙泳划手和换气的配合（陆上模仿练习）

> 视频：P19

**[教学目标]**
- 掌握蛙泳划手的正确动作
- 理解"划→抱→伸"三步口诀
- 为水中练习打下基础

**[教学内容]**

### 19.1 划手动作讲解（5分钟）
- 口诀：**划 → 抱 → 伸**
- 三个动作对应蛙泳划手的三个阶段

### 19.2 划手动作分解（20分钟）
- **划**：
  - 双手往外分开，手掌朝外
  - 手臂弯曲，手掌往外划水
  - 划到肩膀位置
  - 💡 **感觉**：像"抱个大西瓜"，手掌感觉到水的压力

- **抱**：
  - 手臂继续弯曲，往胸口方向抱
  - 手掌从外转成朝后，往身体中线抱
  - 💡 **感觉**：像"抱枕头"，把水往身体这边抱

- **伸**：
  - 手臂伸直，手掌从胸口往前伸
  - 恢复到起始位置
  - 💡 **感觉**：像"扔标枪"，手臂伸直往前冲

### 19.3 陆上模仿练习（20分钟）
- 站在岸上，原地模仿划手动作
- 练习"划→抱→伸"的口令节奏
- 每组20次，休息10秒，重复5组
- ⚠️ **重点**：手不要划太靠后，到肩膀位置就停

### 19.4 划手+换气配合（10分钟）
- 划手的时候抬头换气
- 口令：**划——抱——伸——抬头吸——划——抱——伸——低头呼**
- 练习10组

**[时间分配]**
- 划手讲解：5分钟
- 动作分解：20分钟
- 陆上模仿：20分钟
- 配合练习：10分钟

**[纠错提示]**
| 学员表现 | 纠正方法 |
|------|------|
| 手划太靠后 | 让他"划到肩膀就停，不要继续往后" |
| 划水没力 | 让他"手掌感受水的压力，像抱大西瓜" |
| 换气时手乱动 | 让他"抬头的时候手停在抱的位置" |
| 动作太快 | 让他"慢一点，感受每一个动作" |

## 第21课：蛙泳全身配合

> 视频：P21

**[教学目标]**
- 掌握完整的蛙泳配合
- 能把"划手+换气+蛙泳腿"串联起来
- 能连续游10米以上

**[教学内容]**

### 21.1 配合节奏强化（15分钟）
- 慢动作分解（每个动作单独做）：
  1. **飘着**：身体伸直，往前滑行
  2. **划手**：双手往外划，手掌感受水的压力
  3. **抬头**：划到"抱"的位置，抬头张嘴吸气
  4. **伸手**：手臂往前伸，低头
  5. **收腿**：双手夹耳，头低下，膝盖往胸口收
  6. **翻脚**：脚尖外翻，脚心朝后
  7. **蹬夹**：脚往后蹬，蹬完立刻夹并
  8. **漂**：身体伸直往前滑行

- 💡 **核心理念**：慢！慢！慢！动作越快越容易乱，慢下来才能找感觉

### 21.2 完整配合练习（25分钟）
- **练习一**：慢动作完整配合，每个动作做3秒以上
- **练习二**：正常速度配合，但允许"飘"的时间长一点
- **练习三**：连续配合3次，休息10秒
- **练习四**：连续配合10次，争取游出10米
- ⚠️ **重点**：每组配合之间要"漂"，不能连续做动作不停

### 21.3 常见错误纠正（10分钟）
- ❌ **身体下沉**：蹬腿没用力，或者抬头太高
- ❌ **游得很累**：手脚同时动，或者蹬腿时机不对
- ❌ **不走水**：翻脚不到位，蹬腿没夹水
- ❌ **呛水**：抬头换气太急，嘴巴没张到位

### 21.4 信心建立（5分钟）
- 告诉学员："你已经能游蛙泳了！虽然游得不好，但动作是完整的。"
- 鼓励："继续练习，慢慢就会越来越顺。"

**[时间分配]**
- 节奏强化：15分钟
- 完整配合：25分钟
- 错误纠正：10分钟
- 信心建立：5分钟

**[纠错提示]**
| 学员表现 | 纠正方法 |
|------|------|
| 手脚同时动 | 强调口诀："划手腿不动，伸手再收腿" |
| 游着游着沉了 | 让他"漂的时间再长一点，不要急" |
| 不走水 | 检查翻脚和蹬夹，让他在池边做蛙泳腿 |
| 乱 | 让他"慢下来，一个动作一个动作做" |

- ✅ 知道"慢即是快"的道理

# 🔵 第四阶段：踩水、纠错与提高（P22-P26）
> 核心目标：掌握踩水技能，学会纠错，能够游直线、游得更远

## 第22课：初级踩水在游泳中的应用

> 视频：P22

**[教学目标]**
- 掌握初级踩水的正确动作
- 理解踩水在蛙泳中的实际应用
- 能在游蛙泳时用踩水调整呼吸

**[教学内容]**

### 22.1 踩水动作强化（15分钟）
- 身体直立，肩膀在水面下
- 双手在胸前往下压水，手掌朝下
- 脚在水中像"踩自行车"一样往下踩
- ⚠️ **重点**：头一直在水面上，嘴巴呼吸

### 22.2 踩水与蛙泳的结合（15分钟）
- **练习一**：游蛙泳，游累了就站起来踩水休息
- **练习二**：游蛙泳时偶尔站起来踩水，调整呼吸，再继续游
- **练习三**：尝试"踩水换气"——踩水的时候抬头张嘴吸气

### 22.3 踩水中的换气（10分钟）
- 踩水的时候：用嘴吸气，用鼻子和嘴吐气
- 节奏：吸1-2秒 → 踩水3-4秒 → 吸1-2秒
- 💡 **感觉**：踩水的时候身体像"水上的气球"，有浮力托着

### 22.4 踩水耐力练习（10分钟）
- 连续踩水30秒，休息10秒，重复5组
- 逐渐增加踩水时间到1分钟

**[时间分配]**
- 动作强化：15分钟
- 蛙泳结合：15分钟
- 换气练习：10分钟
- 耐力练习：10分钟

## 第23课：中级踩水及其在游泳中的应用

> 视频：P23

**[教学目标]**
- 掌握更流畅的中级踩水
- 能用踩水维持身体位置
- 能在踩水中完成完整的换气

**[教学内容]**

### 23.1 中级踩水动作（15分钟）
- 与初级踩水的区别：
  - 手压水的幅度更大、更连贯
  - 脚的踩踏更流畅，像"走路"而不是"踩踏板"
  - 身体更稳定，不前后晃

### 23.2 中级踩水练习（15分钟）
- **练习一**：手做"划→抱→伸"的动作来压水（像蛙泳手的简化版）
- **练习二**：脚的踩踏和手的压水配合，形成节奏
- **练习三**：连续踩水1分钟，身体保持稳定

### 23.3 踩水在深水区的应用（10分钟）
- 在深水区无法站立时，用踩水维持身体
- 踩水的同时可以观察四周
- 踩水时可以用嘴吸气，用鼻子和嘴吐气

### 23.4 综合练习（10分钟）
- 游蛙泳 → 踩水调整 → 继续游 → 踩水调整 → 继续游

**[时间分配]**
- 中级踩水动作：15分钟
- 练习：15分钟
- 深水区应用：10分钟
- 综合练习：10分钟

## 第24课：抬头蛙

> 视频：P24

**[教学目标]**
- 掌握"抬头蛙"的动作
- 理解"抬头蛙"与"正常蛙泳"的区别
- 能在抬头蛙中保持前进

**[教学内容]**

### 24.1 抬头蛙的原理（5分钟）
- 正常蛙泳：低头滑行，抬头换气
- 抬头蛙：头一直抬着，嘴巴一直在水面上
- 适用场景：初学者、水浑浊需要观察、踩水不熟练

### 24.2 抬头蛙动作（20分钟）
- **姿势**：头抬着，眼睛往前看，身体大约45度倾斜
- **手**：像正常蛙泳一样划手，但幅度可以小一点
- **腿**：像正常蛙泳一样蹬腿
- **换气**：一直用嘴吸气（因为嘴一直露出水面）
- ⚠️ **注意**：抬头蛙比正常蛙泳累，因为阻力大、无法借滑行的力

### 24.3 抬头蛙练习（20分钟）
- **练习一**：扶浮板，抬头蛙打腿
- **练习二**：不扶浮板，抬头蛙打腿
- **练习三**：抬头蛙完整配合
- 💡 **感觉**：头抬起来看世界，像小青蛙在水面上游

### 24.4 何时用抬头蛙（5分钟）
- 水太浑浊看不清的时候
- 需要观察前方的时候
- 踩水还不熟练的时候
- 正常蛙泳游累了想换换节奏的时候

**[时间分配]**
- 原理讲解：5分钟
- 动作练习：20分钟
- 抬头蛙练习：20分钟
- 何时使用：5分钟

**[纠错提示]**
| 学员表现 | 纠正方法 |
|------|------|
| 身体下沉 | 让他"手多划一下，腿多蹬一下" |
| 游得很累 | 正常现象，告诉他"抬头蛙比正常蛙泳累" |
| 脚沉 | 让他"多蹬腿，蹬完漂一会儿" |
| 乱 | 让他"慢一点，手脚配合好再动" |

## 第25课：游蛙泳总沉下去怎么办？

> 视频：P25

**[教学目标]**
- 诊断"沉下去"的根本原因
- 掌握针对性的解决方法
- 能让自己的蛙泳游得更稳

**[教学内容]**

### 25.1 "沉下去"的原因分析（10分钟）
- **原因一**：蹬腿没力——蹬完腿没有走水，身体下沉
- **原因二**：滑行时间太短——做完动作就急着做下一个
- **原因三**：抬头太高——头抬起来的时候身体就会往下沉
- **原因四**：肺部气太多——憋气太多，身体太重

### 25.2 解决方案（20分钟）
- **方案一**：强化蹬腿力量
  - 翻脚到位：脚尖外翻，脚心朝后
  - 蹬夹连贯：蹬完立刻夹，像剪刀合上
  - 蹬完漂：每蹬一次腿，漂2-3秒

- **方案二**：延长滑行时间
  - 每个周期之间要有"飘"的时间
  - 动作慢一点，不要急

- **方案三**：降低换气高度
  - 嘴巴刚露出水面就吸，不要抬太高
  - 下巴往前伸，不是头往后仰

- **方案四**：吐气充分
  - 每一次低头都要把气吐完
  - 肺部气少，身体才不会太重

### 25.3 分步练习（15分钟）
- **练习一**：深吸气 → 低头 → 慢慢吐完 → 抬头吸 → 低头
- **练习二**：蹬一次腿 → 漂3秒 → 再蹬一次腿
- **练习三**：完整配合，每个动作慢做3秒

### 25.4 自我诊断（5分钟）
- 告诉学员：找出自己"沉"的原因，对症练习
- 记录：每次游泳时注意自己的问题在哪里

**[时间分配]**
- 原因分析：10分钟
- 解决方案：20分钟
- 分步练习：15分钟
- 自我诊断：5分钟

**[纠错提示]**
| 学员表现 | 纠正方法 |
|------|------|
| 沉 | 先问："你是动作一开始就沉，还是游几下后沉？" |
| 如果一开始就沉 | 可能是抬头太高，换气时下巴抬太多 |
| 如果游几下后沉 | 可能是蹬腿没力，或者滑行时间太短 |
| 不确定原因 | 让他做"深吸一口气漂着"的练习，看身体是否水平 |

## 第26课：如何游直线？

> 视频：P26

**[教学目标]**
- 理解"游直线"的重要性
- 掌握让蛙泳游直的方法
- 完成整个蛙泳入门课程

**[教学内容]**

### 26.1 游直线的重要性（5分钟）
- 游直线 = 效率最高 = 最省力 = 游得最快
- 不走直线的常见表现：游到一半撞到泳池壁、S形前进

### 26.2 不走直线的原因（10分钟）
- **原因一**：两边力量不均匀
  - 左手比右手划得多
  - 左腿比右腿蹬得多
- **原因二**：换气时身体歪
  - 总是往一边转头
  - 导致身体向另一边歪
- **原因三**：头部位置不正
  - 头偏向一边
  - 导致身体向另一边偏移

### 26.3 游直线的练习方法（20分钟）
- **方法一**：看泳池底的线，让自己的身体和线平行
- **方法二**：每划一次手，用手指指向正前方，确保方向不变
- **方法三**：换气时两边都要换，不要只换一边
- **方法四**：闭上眼睛游，感受自己的身体是否歪

### 26.4 直线达标测试（15分钟）
- 测试：从泳池一端游到另一端，不碰边
- 标准：身体不歪，能游出相对直的线路
- 记录：游了多少米，用了多少时间（如果能记录的话）

### 26.5 课程总结（5分钟）
- 回顾26课学到的内容
- 告诉学员：蛙泳入门完成，接下来可以：
  - 🎯 继续提高距离（100米、200米）
  - 🎯 学习其他泳姿（自由泳、仰泳、蝶泳）
  - 🎯 学习踩水和踩水技能
  - 🎯 学习出发和转身

## 💡 常见问题与纠正方案

| 问题 | 原因 | 纠正方案 |
|------|------|------|
| **呛水** | 用鼻子吸气 / 换气太急 | 强迫嘴巴吸气、低头慢慢吐气 |
| **漂浮时脚沉** | 憋气太多 / 身体不团 | 吐气、抱膝蜷成一团 |
| **蹬腿不走水** | 翻脚不到位 / 蹬夹不连贯 | 多练翻脚、蹬完立刻夹 |
| **游着游着沉** | 滑行时间太短 / 动作太快 | 慢下来、每组动作之间多漂 |
| **身体歪/S形** | 换气总往一边 / 力量不均 | 两边换气、闭眼游感受歪直 |
| **不往前走** | 翻脚不够 / 蹬水没夹 | 陆上练翻脚、水中练蹬夹 |
| **换气时下沉** | 抬头太高 / 吸气太短 | 下巴贴近水面、嘴巴张大深吸 |
| **很紧张/怕水** | 对水不信任 | 多玩水、建立安全感 |
