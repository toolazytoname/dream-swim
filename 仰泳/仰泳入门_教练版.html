1.1 仰泳的特点（10分钟）

### 1.1 仰泳的特点（10分钟）

  • 仰泳是唯一脸朝上的泳姿——不闷气、不呛水、可以"休息着游"
  • 仰泳的节奏：打腿→划手→换气，循环往复
  • 每个周期中身体有"仰漂休息"的窗口期

1.2 为什么仰泳比蛙泳难？（15分钟）

### 1.2 为什么仰泳比蛙泳难？（15分钟）

  • 仰漂是门槛——初学者不敢仰漂，怕后脑勺磕池底、怕呛水、怕翻过去
  • 换气是难点——吸气在头上方，容易"喝空气"而不是喝水，但本质问题是怕水
  • 身体容易歪——仰卧时没有视觉参照，容易绕圈游

1.3 学习预期建立（10分钟）

### 1.3 学习预期建立（10分钟）

  • 告诉学员：仰泳入门预计需要6-10课时（因人而异）
  • 仰泳比蛙泳"看起来简单"，但基本功要求更高
  • 心理建设：仰漂是学仰泳的第一步，能漂才能游

1.4 仰漂体验（10分钟）

### 1.4 仰漂体验（10分钟）

  • 在浅水区，让学员扶着池边，慢慢往后仰漂
  • 关键：后脑勺要能露出水面——如果后脑勺在水下，说明气没吐完
  • 💡 感觉：像躺在一张床上，水托着背，非常舒服
  • ⚠️ 重点：必须吐气，不能憋气仰漂——憋气会让脚沉下去

2.1 仰卧换气原理（5分钟）

### 2.1 仰卧换气原理（5分钟）

项目时间
仰泳时嘴巴在水面以上，鼻子在水下
换气节奏吸气 → 憋1-2秒 → 呼气（鼻子慢慢吐泡泡）
💡 感觉像闻花香一样吸气，像吹蜡烛一样呼气

2.2 陆上仰卧换气练习（10分钟）

### 2.2 陆上仰卧换气练习（10分钟）

  • 躺在池边或长凳上，头部模拟水中仰卧姿势
  • 练习：吸气 → 转头侧边 → 呼气（感受节奏）
  • 教练喊口令：\"吸——憋——呼——\"
  • ⚠️ 绝对不能用鼻子吸气，否则必呛水

2.3 浅水区仰卧换气（20分钟）

### 2.3 浅水区仰卧换气（20分钟）

  • 练习一：扶池边，仰卧在水面上，嘴巴露出水面，鼻子在水下
  • 嘴巴张开吸气 → 鼻子慢慢呼气
  • 连续10次，组间休息10秒，做3组
  • 练习二：仰卧在浅水区，双手自然放在身体两侧
  • 感受水在脸上的压力——其实并不可怕
  • 💡 感觉：水只是在脸上流过，不会进入嘴巴（因为嘴在水面之上）
  • 练习三：离开池边，原地踩水同时仰卧换气
  • 保持嘴巴一直在水面上方

2.4 仰漂换气进阶（15分钟）

### 2.4 仰漂换气进阶（15分钟）

  • 抱膝仰漂 → 双手放开 → 仰卧漂浮
  • 嘴巴吸气 → 鼻子呼气 → 身体保持稳定
  • 💡 感觉：脸在水面上方，空气从嘴巴进来，从鼻子出去
  • ⚠️ 关键：呼气要"轻而长"，不能"噗"地一下把气喷出来——那样会让自己紧张

第3课：仰泳的腿部动作
核心课
🎯

3.1 仰泳打腿原理（10分钟）

### 3.1 仰泳打腿原理（10分钟）

  • 仰泳腿和自由泳腿类似，都是鞭状打水
  • 关键：大腿发力 → 膝盖放松 → 小腿和脚自然鞭打
  • 口诀：上踢直，下打弯，腿打水像鱼尾巴
  • 上踢：腿向上踢时，脚踝伸直，踢出水面
  • 下打：腿向下打时，膝盖微弯，用脚背推水

3.2 陆上仰泳打腿模仿（10分钟）

### 3.2 陆上仰泳打腿模仿（10分钟）

  • 仰卧在池边，两手撑身体
  • 练习打腿动作：慢上快下，感受鞭状
  • 💡 感觉：像用鞭子抽水——鞭子头（脚）是最后动的
  • ⚠️ 常见错误：膝盖打得太大（像蛙泳腿）、脚踝不放松

3.3 扶池边打腿练习（15分钟）

### 3.3 扶池边打腿练习（15分钟）

  • 双手扶池边，身体仰卧在水面
  • 练习一：轻轻打腿，保持身体稳定
  • 练习二：打腿时感受"打上来的水花"——有力但不猛
  • 练习三：连续打腿30秒，休息10秒，重复3组
  • 💡 感觉：脚踝像绑着带子一样甩动，力量从大腿传到脚踝

3.4 仰漂打腿（15分钟）

### 3.4 仰漂打腿（15分钟）

  • 蹬边滑行 → 进入仰漂 → 开始打腿
  • 目标：打腿时身体保持水平，不歪斜
  • ⚠️ 重点：
  • 头保持正中，眼睛看天花板——不要转头
  • 肩膀不要用力，放松
  • 脚踝放松，像鱼尾一样鞭打
  • 💡 感觉：身体像一根木棍在水中旋转，腿在下面打水

### 4.1 配合节奏讲解（5分钟）

4.2 仰漂打腿+换气练习（25分钟）

### 4.2 仰漂打腿+换气练习（25分钟）

  • 练习一：仰漂打腿，每3秒换一口气
  • 打3下腿 → 嘴张开吸气 → 继续打腿

  • 练习二：仰漂打腿，吸气后"憋一下"再呼气
  • 吸一口气 → 默数3 → 鼻子慢慢呼气
  • 💡 感觉：换气时身体不要起伏，像一块木板在水上漂
  • 练习三：连续游动——仰漂打腿+换气配合
  • 打腿保持稳定，吸气保持节奏
  • 目标：能连续游10米以上

5.1 转体原理（10分钟）

### 5.1 转体原理（10分钟）

  • 仰泳时每划一次手，身体会自然转向一侧
  • 转体的作用：
  • 换气：转头侧边，嘴巴露出水面吸气
  • 复位：让身体回到正中位置
  • 口诀：划手转体，头随之动，嘴露出水，吸气不停

5.2 转体换气练习（20分钟）

### 5.2 转体换气练习（20分钟）

  • 练习一：仰漂打腿，单手扶池边
  • 另一只手划手，身体转向一侧
  • 转头，嘴巴露出水面，吸气
  • 继续打腿，身体回到仰卧
  • 练习二：自由转体（不扶池边）
  • 打腿 → 划手 → 转体 → 吸气 → 复位
  • 连续练习10次，休息10秒
  • 练习三：转体+换气+打腿完整配合
  • 每划一次手，换一次气
  • 划左臂转左侧吸气，划右臂转右侧吸气
  • 💡 感觉：转体时像"拧毛巾"——身体拧过去再拧回来

### 6.3 双边划手练习（15分钟）

7.1 转身原理（10分钟）

### 7.1 转身原理（10分钟）

项目时间
前滚翻转身将仰卧姿态变成俯卧姿态，再变成仰卧姿态
目的在泳池尽头改变方向继续游
关键触壁 → 收腹 → 翻过去 → 蹬壁

7.2 陆上转身模仿（10分钟）

### 7.2 陆上转身模仿（10分钟）

  • 仰卧在地上，模拟翻滚动作
  • 练习：收腹→低头→翻过去→屈膝→站起
  • 💡 感觉：像球一样滚过去，身体要蜷缩

7.3 浅水区转身练习（20分钟）

### 7.3 浅水区转身练习（20分钟）

  • 练习一：扶池边翻转
  • 双手扶池边，身体仰卧 → 收腹 → 翻过去
  • 身体从仰卧变成俯卧，再变回仰卧
  • 练习二：水中翻转（不触壁）
  • 仰漂 → 收腹 → 低头 → 翻过去
  • 身体滚一圈，回到仰卧姿态
  • 练习三：完整前滚翻转身
  • 游到池边 → 触壁 → 收腹 → 翻过去 → 屈膝 → 蹬壁

7.4 蹬壁滑行（5分钟）

### 7.4 蹬壁滑行（5分钟）

  • 转身完成后，身体蜷缩在池壁旁
  • 双脚踩池壁，用力蹬出去
  • 进入仰漂滑行，继续游

⚠️ 安全提示：前滚翻时头部可能磕到池壁，教练要提前告知学员
